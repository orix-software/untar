# Command: untar

### untar utility

## SYNOPSYS
+ untar -t[v]f tarfile
+ untar -x[v]f tarfile

## EXAMPLES
+ untar -tf /home/test.tar

## DESCRIPTION
**untar** list and extract files in a TAR archive

## OPTIONS
*  -h
                show this help message and exit
*  -t
                list files
*  -x
                extract files
*  -v
                verbose mode

## SOURCE
https://github.com/orix-software/untar
